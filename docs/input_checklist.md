# Input Checklist

These are the exact inputs that will unlock the next useful build step.

## Trusted sources

For each expert or firm:

- name
- why you trust them
- priority: core, secondary, or experimental
- market focus
- every publishing channel and link
- whether the source is public, paid, or private

## Markets to support first

- core market buckets
- specific commodities or sub-assets
- preferred holding horizon

## Data and credentials

- OpenAI API key for model-backed extraction
- X or approved social ingestion provider
- YouTube API key if video channels matter
- transcript provider if podcasts are important
- market data provider for validation inputs
- any private newsletter or feed access workflow

## Output preferences

- weekly memo, daily brief, alerts, or all three
- what “positioning” should mean in the report
- the level of aggressiveness versus caution in recommendations
